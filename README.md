# DrumRoll
A Js Experiment for a Drum Sound Kit

## How to Play:
It is a simple kit for hitting buttons on the page or keys of the keyboard, to make the drum sounds.
- W for Tom1
- A for Tom2
- S for Tom3
- D for Tom4
- J for Snare
- K for Crash
- L for Kick
